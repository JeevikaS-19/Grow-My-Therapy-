
# Lilac Template - Private Practice Website

A high-end, responsive landing page and blog template designed specifically for wellness professionals and private therapy practices. This project focuses on a "sanctuary-like" user experience through minimalist design, breathable white space, and elegant typography.

## Features

- **Modern Architecture**: Built with React 19 and Next.js-style component organization.
- **Fluid Design**: Fully responsive across mobile, tablet, and desktop.
- **Micro-Interactions**: Smooth scroll-reveal animations and transition effects.
- **Editorial Typography**: Uses a sophisticated mix of Playfair Display and Inter.
- **Custom Aesthetic**: Unique arched imagery and circular framing to evoke a sense of calm and growth.
- **Functional Components**: Includes interactive FAQ accordions, professional background toggles, and multi-page routing for Blog and Contact sections.

## Tech Stack

- **React 19**: Core UI logic.
- **Tailwind CSS**: Utility-first styling for precise layout control.
- **Native Intersection Observer API**: For high-performance scroll animations.
- **ES6 Modules**: Clean, modular code structure.

## Development

The project is structured into modular components located in the `components/` directory. Styles are managed primarily through Tailwind utility classes with a few custom CSS variables for the core brand palette defined in `index.html`.
